# TicTacToe game

This is a project meant to reacreate a console version of tic-tac-toe using c++

## Build

For now building is only available on linux.
* Prerequisites
    > cmake
    > build-essential
    > g++

    In order to build just type:
    ```
    make build
    ```

## Playing
After building just type:
```
./bin/tictactoe
```
